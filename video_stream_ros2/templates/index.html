Website Stream of Webcam

Description of the Projetct

The object of this project is to make a website stream of camera image.

Subscriber

The subscriber is a Python code (app.py) that recieves the message from the topic /image and share it in a hosted website using the Flask Framework. As the topic message can arrive at any time and the Flask application cannot stop, events related to ROS are executed in parallel within the code, through Threads. The trick is create the subscriber node and target it in the Thread. This makes any event in the node redirect for the image processing functions, like a external interrupt in microcontrolers. The subscriber node name is /Show_image_python.
